# 1. Анализ строк A-H

| Строка | Компилируется? | Почему?                                                                                                                                               |
| :----- |:---------------|:------------------------------------------------------------------------------------------------------------------------------------------------------|
| A      | Да             | `name` имеет модификатор `public`, доступен из любого места.                                                                                          |
| B      | Нет            | `age` имеет модификатор `protected`, доступен только в том же пакете или наследникам. `HRSystem` находится в другом пакете и не наследует `Employee`. |
| C      | Нет            | `salary` имеет модификатор доступа по умолчанию (package-private), доступен только внутри пакета `company.core`.                                      |
| D      | Нет            | `password` имеет модификатор `private`, доступен только внутри класса `Employee`.                                                                     |
| E      | Да             | `getRole()` имеет модификатор `public`, доступен из любого места.                                                                                     |
| F      | Нет            | `promote()` имеет модификатор `protected`, доступен только в том же пакете или наследникам.                                                           |
| G      | Нет            | `printSummary()` имеет package-private доступ.                                                                                                        |
| H      | Нет            | `validatePassword()` имеет модификатор `private`, доступен только внутри класса `Employee`.                                                           |
