Программы, материалы и задачи второго семестра второго курса по дисциплине "Современные технологии программирования".

Горячие клавиши

Работа с кодом:
- Ctrl + Space - базовая автоподстановка.
- Ctrl + D - дублировать текущую строку.
- Ctrl + Y - удалить текущую строку.
- Ctrl + Alt + L - автоформатирование.

Работа с Git:
- Ctrl + K - открыть окно Commit.
